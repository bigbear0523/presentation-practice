# プレゼン暗記練習アプリ

プレゼンの台本を貼り付けて、暗記練習ができるWebアプリです。  
ブラウザだけで動作し、データは外部に送信されません。

## 起動手順

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動（ブラウザが自動で開きます）
npm run dev
```

ブラウザで `http://localhost:3000` が開きます。

## 主要機能

### 台本入力
- テキストエリアに台本をコピペ
- 「反映する」ボタンで練習用データに変換
- サンプル台本が最初から入っています

### 分割表示
- **1文ごと**: 句点「。」「！」「？」や改行で分割
- **段落ごと**: 空行で分割
- 長すぎる文は読点「、」で補助分割

### 読み上げ機能（SpeechSynthesis API）
- 現在の文を読む / 全文を順番に読む
- 一時停止・再開・停止
- 速度調整（0.6x〜2.0x）
- 読み上げ中はカードが青く光ります
- 日本語音声を自動的に優先選択

### 練習モード

| モード | 説明 |
|--------|------|
| 通常表示 | 全文をそのまま表示 |
| 穴埋め練習 | 文を隠して、思い出してから表示 |
| 先頭ヒント | 最初の数文字だけ表示（文字数は調整可） |
| 自動送り | 数秒ごとに次の文へ自動で進む（読み上げ付き） |

### 操作
- 前へ / 次へ / 最初へ / ランダム
- 表示/非表示の切替
- 現在位置の表示（例: 3/18）
- 進捗バー

### 暗記サポート
- チェック済みマーク
- 苦手マーク → 苦手だけ練習
- 進捗一覧で全体を確認
- localStorageで台本・進捗・苦手マークを保存
- ページ再読み込み後も状態を保持

### UI
- ライト/ダークモード切替
- レスポンシブ対応（PC推奨、スマホでも利用可）
- 大きめ文字で視線移動を少なく

## 技術構成

- React 18 + TypeScript
- Vite
- CSS変数によるテーマ
- SpeechSynthesis API（ブラウザ標準）
- localStorage（データ保存）
- 外部API・外部通信なし
