# 営業指導 四半期振り返り・フィードバックアプリ

郵便局の営業指導業務を効率化するための、四半期振り返りデータ管理・見える化アプリです。

## 機能概要

- **社員管理**: 指導対象社員の登録・編集・削除
- **四半期データ入力**: 活動実績・自己評価・自由記述の入力
- **社員詳細**: 個別の履歴・グラフ・成長の軌跡表示
- **フィードバック**: 自動フィードバック補助・前回比較・印刷用レイアウト
- **年間サマリー**: 年間推移・フィルタ・CSVエクスポート

### 見える化機能
- 活動実績推移（棒グラフ・折れ線）
- 自己評価レーダーチャート
- 自己評価と指導者評価のギャップ表示
- 前回四半期との比較
- 強み・弱み上位3項目
- 活動量と自己評価のバランス
- 指導優先ポイント表示
- ルールベース自動フィードバック補助

## 起動方法

### 1. 環境準備

```bash
cd quarterly-review-app
pip install -r requirements.txt
```

### 2. アプリ起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が開きます。

### 3. サンプルデータ

初回起動時に4名分のサンプルデータが自動投入されます。

## ディレクトリ構成

```
quarterly-review-app/
├── app.py                     # ダッシュボード（トップ画面）
├── pages/
│   ├── 01_社員管理.py          # 社員一覧・登録・編集・削除
│   ├── 02_四半期入力.py        # 四半期データ入力フォーム
│   ├── 03_社員詳細.py          # 社員別詳細・グラフ・履歴
│   ├── 04_フィードバック.py     # フィードバック画面・印刷用
│   └── 05_年間サマリー.py      # 年間集計・CSV出力
├── lib/
│   ├── database.py            # SQLite操作（CRUD・サンプルデータ）
│   ├── charts.py              # Plotlyグラフ生成
│   ├── feedback_engine.py     # ルールベース自動フィードバック
│   └── export.py              # CSVエクスポート
├── data/
│   └── quarterly_review.db    # SQLiteデータベース（自動生成）
├── requirements.txt
└── README.md
```

## データ管理

- **保存形式**: SQLite（`data/quarterly_review.db`）
- **テーブル構成**: `employees`（社員マスタ）+ `quarterly_reviews`（四半期データ）
- **バックアップ**: `data/quarterly_review.db` ファイルをコピーするだけでOK
- **CSVエクスポート**: 年間サマリーページからダウンロード可能

## 注意事項

- 顧客情報（氏名・証券番号・契約番号等）は入力しないでください
- 本アプリは営業指導用の内部振り返り専用です
- ローカル環境での使用を前提としています
